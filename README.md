# DinoGame (Logisim Version)

A simple dinosaur game designed using **Logisim**, demonstrating basic digital logic for a game.

## Files

## How to Run

1. Install [Logisim](http://www.cburch.com/logisim/).  
2. Load all the files and simulate the game using switches and LEDs.

## Features

- Jump and obstacle logic using digital circuits  
- Scoring using counters  
- Combines combinational and sequential logic  

## Learning Outcomes

- Practice with logic gates, flip-flops, and counters  
- Understanding of digital game design using circuits
